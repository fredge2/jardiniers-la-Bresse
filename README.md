# Site Web des Jardiniers de La Bresse

## 🌱 Présentation

Site web officiel de l'association des Jardiniers de La Bresse, créé pour célébrer les 30 ans de l'association (1995-2025).

## 📁 Structure du Site

### Pages Principales
- **index.html** - Page d'accueil avec présentation générale
- **association.html** - Histoire et valeurs de l'association
- **activites.html** - Activités, formations et jardins partagés
- **30ans.html** - Programme spécial des 30 ans
- **contact.html** - Informations de contact et formulaire

### Dossiers
- **css/** - Feuilles de style
- **js/** - Scripts JavaScript
- **images/** - Images et illustrations

## 🚀 Installation et Utilisation

### Utilisation Locale
1. Décompressez l'archive
2. Ouvrez `index.html` dans votre navigateur
3. Naviguez entre les pages via le menu

### Déploiement Web
1. Uploadez tous les fichiers sur votre serveur web
2. Assurez-vous que la structure des dossiers est préservée
3. Le site est prêt à l'emploi !

### Personnalisation
- Modifiez les couleurs dans `css/styles.css` (variables CSS)
- Remplacez les images dans le dossier `images/`
- Adaptez le contenu dans les fichiers HTML

## 📧 Formulaire de Contact

Le formulaire de contact inclut :
- Validation côté client
- Champs obligatoires marqués
- Messages d'erreur informatifs
- Simulation d'envoi (à connecter à un service d'email)

### Pour activer l'envoi d'emails
Intégrez un service comme :
- EmailJS
- Formspree
- Netlify Forms
- Ou votre propre backend
